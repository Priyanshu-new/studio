# **App Name**: Abily Learn

## Core Features:

- AI Assistant (Abily AI): Multi-modal AI assistant with voice, text, and visual input, providing real-time tutoring and information, including input from Sign Language to English conversion.
- Adaptive Interface: Switchable modes and structured plane mode for different disabilities (hearing, visual, cognitive) and needs.
- Sign Language â†” English Conversion: Converts Indian Sign Language (ISL) to English and vice versa using live camera input. The AI assistant can use the output of this feature.
- Facial Expression & Gesture Detection: Recognizes sign language, gestures, and facial expressions via camera input to trigger actions such as music play, controlled by the LLM with reasoning using a tool.
- Quiz Generator: Automatically generates quizzes from selected topics, providing graded feedback, explanations, and hints. Includes support for the AI assistant tool for clarifications.
- Chunk-Based Learning: AI splits large content into understandable chunks and summarizes long content for better comprehension.
- User Authentication & Personalization: User login with profile settings (disability type, language preference), storing history and progress to adapt content.
- Offline-First Architecture: Enables content and assistant functionality even in low or no internet conditions, with syncing when online.

## Style Guidelines:

- Primary color: Soft blue (#75A3D1) to promote a sense of calmness and focus suitable for a learning environment.
- Background color: Very light blue (#F0F4F8), complementing the primary color for a soothing and distraction-free backdrop.
- Accent color: Warm yellow (#E5B85C) for interactive elements and highlights to draw attention and create a friendly interface.
- Font: 'PT Sans' (sans-serif) for clear and accessible body text, ensuring readability for all users.
- Font: 'Space Grotesk' (sans-serif) for headlines to create a computerized, techy, scientific feel.
- Use clear, simple, and universally recognizable icons with high contrast to aid users with cognitive and visual impairments.
- Implement a clean and structured layout with ample whitespace to minimize visual clutter and improve focus, especially beneficial for users with cognitive disabilities.
- Use subtle and purposeful animations to guide user interaction and provide feedback without causing distraction or overwhelming users with sensory sensitivities.
